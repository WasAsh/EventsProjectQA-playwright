# EventsProjectQA-playwright

Developed a simple automation framework for the EventHub web application using Playwright and TypeScript.


*Responsibilities*

-Designed scalable Page Object Model architecture.
-Developed reusable UI automation components.
-Automated end-to-end user journeys.
-Created Login, Registration, Event Booking and Booking Verification sample tests.
-Integrated automated tests with GitHub Actions.
-Improved locator stability and reduced flaky tests.
-Implemented assertion utilities and helper methods.
-Performed API validation.
-Generated HTML test reports.
-Allure test report.
-Integrated with Jenkins.
